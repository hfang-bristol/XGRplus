Section 11 SUBNETWORK ANALYSER (GENES)

11.1 Input Interface

Input

  • Step 1: User-input gene list, with 1st column for gene symbols and 2nd column for the significance info (p-values between 0 and 1).

  • Step 2 and Step 3: The subnetwork of highly-scored genes is to be identified from which gene network (Step 2), with the desired number of genes in the subnetwork (Step 3). The significance (p-value) of observing the identified subnetwork by chance can be also estimated by a degree-preserving node permutation test.

  • More Controls: More parameters to fine-tune steps described above.

Output

  • Example Output includes interactive tables nd and high-quality figures for subnetwork results. A summary of input data and the runtime (computed on the server side) is also returned for the reference.
